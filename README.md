# News App

Basic News app made with React Native. 

### Features:
- OTP Login
- Add news as Favorites
- Profile
- News in PDF
- Videos
- Notifications
- News Comments
- Multi Language
- Light / Dark Theme

### Technologies
- React Native
- React
- Expo
- Redux
- AsyncStorage
- UI Kitten
- Moment
- Firebase
- Axios
- Node js Backend
